# # 💰 Client Payment Manager

## 📖 Overview

Client Payment Manager is a simple web-based application developed using **HTML, CSS, and JavaScript**. It helps users maintain client bank details, enter payment amounts, search client records, and print selected payment information in a clean, organized format.

The application uses the browser's **Local Storage** to save client data, allowing records to remain available even after refreshing the page.

---

## 🎯 Purpose

Managing client payment details manually can be time-consuming and error-prone. This application provides a simple digital solution to:

- Store client banking information.
- Quickly search client records.
- Enter payment amounts before processing payments.
- Print selected client details for record keeping.
- Automatically calculate the total payment amount of selected clients.

---

## ✨ Features

- Add new client details.
- Store bank name, branch, account number, and IFSC code.
- Search clients instantly.
- Enter payment amount for each client.
- Select multiple clients using checkboxes.
- Print selected client payment details.
- Automatically calculate the grand total of selected payments.
- Delete client records.
- Automatically save data using Local Storage.

---

---

## 🚀 How to Use

1. Open the **index.html** file in any modern web browser.
2. Enter the client's:
   - Name
   - Bank Name
   - Branch
   - Account Number
   - IFSC Code
3. Click **Add Client** to save the details.
4. Use the search bar to find clients quickly.
5. Enter the payment amount for the required clients.
6. Select one or more clients using the checkboxes.
7. Click **Print Selected** to generate a printable payment sheet.
8. The application automatically displays the **Grand Total** of all selected payment amounts.
